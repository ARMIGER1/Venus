link ( $  title = 'New Link',
$  url = '#',
$  target = '_self' 
)

Adds a link to the page.

Parameters:
string$titleThe title of the new link as displayed on the page.
url$urlThe URL of the page being linked to.
string$targetThe specified target of the link.

$target may have 1 of 4 parameters:

  1. _blank - Opens the link in a new window or tab.
  2. _self - Opens the link in the same frame as the link that was clicked. DEFAULT
  3. _parent - Opens the link in the parent frameset.
  4. _top - Opens the link in the full body of the window.
